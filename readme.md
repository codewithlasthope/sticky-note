# 📝 Sticky Notes App

A simple and lightweight **Sticky Notes App** built with **HTML**, **CSS**, and **Vanilla JavaScript**. Create notes instantly, edit them in real time, and automatically save them using your browser's Local Storage.

## 🚀 How It Works

1. Click the **Add** button to create a new sticky note.
2. Type your text inside the note.
3. Every change is automatically saved to Local Storage.
4. Refreshing the page restores all previously saved notes.
5. Double-click any note to delete it.
6. Press **Enter** to exit editing.

---

## 💾 Data Storage

All notes are stored locally in your browser using the **Local Storage API**.

Example data format:

```javascript
[
  "Buy groceries",
  "Complete JavaScript project",
  "Meeting at 5 PM"
]
```

No backend, database, or server is required.
